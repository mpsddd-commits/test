# 부트스트랩 관련 클래스
## 글자관련 클래스
| 목적  | 클래스                                         |
| --- | ------------------------------------------- |
| 색상  | `text-primary`, `text-danger`, `text-muted` |
| 굵기  | `fw-bold`, `fw-semibold`                    |
| 정렬  | `text-center`, `text-end`                   |
| 크기  | `h1~h6`, `display-*`                        |
| 스타일 | `fst-italic`                                |
| 반응형 | `text-md-start`                             |
